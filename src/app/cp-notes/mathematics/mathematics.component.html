Game Theory

  • Sprague-Grundy Theorem -

    For a nim game, if the cumulative xor of the grundy numbers of each game is non-zero, the first player will win, else if the second player will.

Number Theory

  • Euler's totient function -

    φ(x) is the number of integers less than x that are coprime to x

    φ(x) is equal to x * product of (p-1)/(p), where p is every distinct prime factor of x

    ie, φ(24) = 24 * (1/2) * (2/3) = 8

  • Euler's theorem of modular exponents -

    When calculating modular powers, ab is congruent to aφ(m) + b%φ(m), mod m

    ie, 1420 % 24 = 1412 % 24 = 16
